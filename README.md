# ✅ Data Validation Project

 “A responsive data validation project built using HTML, CSS, and JavaScript with regex-based form validation for user inputs.” 
This project uses JavaScript Regex (Regular Expressions) to validate user inputs such as name, email, and mobile number in real time.

## 🚀 Features
- Real-time form validation
- Regex-based input checking
- Name validation with minimum character rules
- Mobile number validation (10 digits)
- Email format verification
- Clean and responsive UI
- Beginner-friendly frontend project

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Regular Expressions (Regex)

## 📌 Validation Rules
- Name must contain at least 6 characters
- Mobile number must contain exactly 10 digits
- Email must follow a valid format (`example@email.com`)

## 📌 Project Purpose
This project was created to improve understanding of JavaScript form handling, Regex concepts, DOM manipulation, and frontend validation techniques.
